Required Tools:

Instructions: 1. Accept the repair order by phone: Press the blue button. The button will display the type of repair order (general repair, electrical repair, plumbing repair). 2. Tool selection: Simultaneously, the table below will display the tools you need to take with you. The border around the table will turn orange. 3. Place tools in the basket: The type of the current repair will be displayed on the right button. Press the button to put the selected tools into the basket. The border around the tools will turn green (indicating the tools are in the basket), and you can proceed to the order. 4. Accept the next call: If the repair type is the same (e.g., it was an electrical repair and the new order is also for electrical repair), proceed to the new/next order. If the repair type is different (e.g., it was an electrical repair, and the new order is for plumbing repair), return for a new set of tools. 5. Total Service Calls table: You can see the counters of repair orders since the start of the program.
